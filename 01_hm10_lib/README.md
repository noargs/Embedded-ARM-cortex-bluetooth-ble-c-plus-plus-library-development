### UART GPIO Config for NucleoF411
      
<u>UART1 to use with HM10 </u>       	    
         
* We use **PA9** as `UART1_Tx`
* And **PA10** as `UART1_Rx`
       
       
<u>UART2 to use for Debug output</u>      
       
* We use **PA2** as `UART2_Tx`
* And **PA3** as `UART2_Rx`
